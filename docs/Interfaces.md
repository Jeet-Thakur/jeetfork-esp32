# Sensors and LEDs
- LED1 - GPIO33
- LED2 - GPIO4
- Switch (SW) - GPIO32
- LDR (Light Dependent Resistor) - GPIO39
- Potentiometer (POT) - GPIO36

# Interfaces
## UART
- Rx - GPIO17
- Tx - GPIO16
## I$^2$C
- SDA - GPIO21
- SCL/SCK - GPIO22
## CAN
- Rx - GPIO26
- Tx - GPIO25
## SPI
- CS - GPIO15
- DO - GPIO12
- DI - GPIO13
- CLK - GPIO14
